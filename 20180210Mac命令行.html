Mac系统常用命令行

1,系统操作

Finder显示隐藏文件

在不安装第三方增强软件的前提下，Finder 这个系统级的程序并没有提供将隐藏文件显示出来的按钮，但这个功能是有的，只不过隐藏太深，一般这类文件是不需要你去维护的，如果你非要找到他们，可以看以下步骤：

1,进入要显示隐藏文件的目录里；

2,按组合键 Command+F 显示搜索栏；

3,将搜索属性从“种类”改为“其他…”；

4,在「选择搜索属性」菜单里，直接搜索并找到 “文件不可见” 这一项；

5,勾选其最右侧确认框，点击“好”

6,这时你可以返回搜索属性菜单，选择新出现的“文件不可见”，值选择“不可见项目”

7,此时隐藏文件就显示出来了

如果你想一劳永逸，永久显示这些隐藏文件，可以在终端里执行：

        defaults write com.apple.finder AppleShowAllFiles -bool true
    //重新隐藏他们可执行：
        defaults write com.apple.finder AppleShowAllFiles -bool false
    //最后再重启 Finder 即可

几个有用的命令行清理命令

        删除 QuickLook 缓存文件：sudo rm -rf /private/var/folders/
        删除 Emacs：sudo rm -rf /usr/share/emacs/
        删除系统缓存： ” cd /private/var/tmp/ ” rm -rf TM*
        删除系统日志：sudo rm -rf /private/var/log/*

显示资源库

方法一：

打开Finder，菜单中选择前往按住option键就会显示资源库项(每次打开都需要重复操作一次)。

方法二:

在“终端”中输入下面的命令：
        chflags nohidden ~/Library/
        //隐藏
        chflags hidden ~/Library/
