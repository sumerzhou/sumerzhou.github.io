20170320-Node扩展框架express

1，express概述

express是一个基于node.js平台的极简、灵活的web应用开发框架，它提供一系列强大的特性，帮助创建各种web和移动设备应用。

它不对node.js已有的特性进行二次抽象，只在它之上扩展web应用所需的基本功能。

中文官网

2,安装

初始化仓库(npm init):

创建文件夹 -> 在文件夹下打开命令窗口 -> 输入命令npm init 初始化仓库 ->配置项目文件:

  • 生成一个package.json文件
  • name:项目名称
  • version:版本号
  • description:项目描述
  • entry point:(index.js):当前应用的入口文件
  • author:作者
  • 其他可以默认:回车

安装express:

接着在命令窗口输入 npm install express进行安装。

安装依赖:npm install express --save 。指定--save 参数,此模块将被添加到package.json文件中dependencies依赖列表中。若无法安装,使用淘宝镜像。

3,使用

        var express = require('express');
        var app = express();
        app.get('/',function(req,res){
            res.send('Hello Express!');
        });
        //post方法:
        app.post('/',function(req,res){
            console.log('post...);
            res.setHeader('Content-Type','text/html');
            res.send('<h1>Hello Express post!</h1>');
        });
        var server = app.listen(3000,function(){
            var host = server.address().address;
            var port = server.address().port;
            console.log('Listening at http://%s:%s',host,port);
        });

加载文件:

使用sendFile,需要把html文件放到映射文件夹内(如放到pubilc内)。但此方法加载出的文件是静态文件。

        app.get('/index.html',function(req,res){
            res.sendFile('index.html');
        });

或,使用fs读取文件:

        var fs = require('fs');
        app.get('/index.html',function(req,res){
            fs.readFile('index.html',function(err,data){
                if(err){
                    fs.readFile('404.html',function(err,data){
                        if(err) throw err;
                        res.setHeader('Content-Type','text/html');
                        res.end(data);
                    });
                }else{
                    res.setHeader('Content-Type','text/html');
                    res.end(data);
                }
            });
        });

post提交的数据处理,可去官网查看方法。有一个"中间件"处理的方法(body-parser req.body)。

4,文件访问

服务器的本地文件属于网络资源，想要访问需要映射到网络上，express提供了中间件可以直接操作。

app.use(express.static('puclic'));访问public下的文件
