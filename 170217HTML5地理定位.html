20170217-HTML5地理定位

1,HTML5地理定位简介

HTML5 Geolocation（地理定位）用于定位用户的位置。

HTML5 Geolocation API(地理位置应用程序接口) 用于获得用户的地理位置。鉴于该特性可能侵犯用户的隐私，除非用户同意，否则用户位置信息是不可用的。

对于拥有 GPS 的设备，比如 iPhone，地理定位更加精确。

geolocation API存在于navigator对象中，只包含3个方法：getCurrentPosition()获取当前位置、watchPosition（） 监视位置持续获取当前用户位置、clearWatch（） 清除监视。

2,HTML5地理定位代码实现

检测浏览器是否支持：
        if (navigator.geolocation) {
            //console.log("浏览器支持!");
            navigator.geolocation.getCurrentPosition(showPosition);
         }
        else {
            // console.log("浏览器不支持!");
         }

getCurrentPosition(onSuccess,onError,options),onSuccess方法成功时调用的(必选)，onError方法失败是调用的(可选)，options其他参数(可选)。如果getCurrentPosition()运行成功，则向参数showPosition中规定的函数返回一个coordinates对象。通过这个对象可以获取地理位置的相关信息:

成功时：
        function showPosition(position){
               //返回用户位置
               //经度
               var longitude =position.coords.longitude;
               //纬度
               var latitude = position.coords.latitude;

               //使用百度地图API
               //创建地图实例
               var map =new BMap.Map("container");  //地图放在id为container的区域
               //创建一个坐标
               var point =new BMap.Point(longitude,latitude);
               //地图初始化，设置中心点坐标和地图级别
               map.centerAndZoom(point,15);
        }

若成功，则 getCurrentPosition() 方法返回对象。始终会返回 latitude、longitude 以及 accuracy 属性。如果可用，则会返回其他下面的属性:

  • coords.accuracy:位置精度
  • coords.altitude:海拔，海平面以上以米计
  • coords.altitudeAccuracy:位置的海拔精度
  • coords.heading:方向，从正北开始以度计
  • coords.speed:速度，以米/每秒计
  • timestamp:响应的日期/时间

getCurrentPosition() 方法的第二个参数用于处理错误。它规定当获取用户位置失败时运行的函数：

失败时:
        function showError(error){
          switch(error.code){
            case error.PERMISSION_DENIED:
              x.innerHTML="User denied the request for Geolocation."
              break;
            case error.POSITION_UNAVAILABLE:
              x.innerHTML="Location information is unavailable."
              break;
            case error.TIMEOUT:
              x.innerHTML="The request to get user location timed out."
              break;
            case error.UNKNOWN_ERROR:
              x.innerHTML="An unknown error occurred."
              break;
            }
          }
错误代码：Permission denied-用户不允许地理定位,Position unavailable-无法获取当前位置,Timeout-操作超时。

3,在地图中显示结果(百度地图API简单使用)

如需在地图中显示结果，您需要访问可使用经纬度的地图服务，比如谷歌地图或百度地图。

百度地图API是由JavaScript语言编写的，在使用之前需要将API引用到页面中。(实时定位地图)

更多请参考百度开放文档：http://developer.baidu.com/map/jsdemo.htm#a1_2
