20170111-JS原型与原型链

1，原型与原型链概念

涉及到继承这一块，Javascript 只有一种结构，那就是：对象。在 javaScript 中，每个对象 都有一个指向它的原型（prototype）对象的内部链接。这个原型对象又有自己的原型，直到 某个对象的原型为 null 为止（也就是不再有原型指向），组成这条链的最后一环。这种一级 一级的链结构就称为原型链（prototype chain）。

在javascript中，每个对象都有一个与之相关联的对象，那就是它的原型对象。所有实例对象都 从它的原型对象上继承属性。

2，原型对象与实例对象

以构造函数创建对象为例：

构造函数拥有一个prototype属性，指向原型。一个对象的原型就是它的构造函数的prototype属性的值。 

          function Person(name,age) {
               this.name = name;
               this.age = age;
          }
          console.log(Person.prototype);  //object。当一个函数被定义时，会自动创建和初始化
                                            prototype值，这个值是一个对象。
          console.log(Person.prototype.constructor == Person);  //true。这个对象只有一个属
                                           性constructor，它指回和原型相关联的那个构造函数。
          var student =new Person();
          console.log(student.__proto__===Person.prototype);//true。实例内置原型指针指向构
                                                             造函数的原型对象。
          console.log(student.__proto__.constructor==Person);//true

通过原型来创建属性和方法，可被所有实例所共享。若在一个实例中修改了该属性或方法的值，所有其他实例的属性或方法值都会受到影响。

          function Person(name,age){
            this.name=name;
            this.age=age;
          }
          Person.prototype.friends=["Tom","Sam"];
          var p1=new Person("Lily",24);
          var p2=new Person("Susan",20);
          alert(p1.friends); //Tom,Sam
          alert(p2.friends); //Tom,Sam
          p1.friends.push("Bill");
          alert(p1.friends); //Tom,Sam,Bill
          alert(p2.friends); //Tom,Sam,Bill

3，注意

如果原型对象被赋值为一个对象时相当于重写了整个原型对象，因此切断了它与构造函数的关系，此时Person.prototype.constructor不再指向Person。 

           function Person(){  }
           Person.prototype={
              name:"Tom",
              age:29
          }
          var p1=new Person();
          alert(Person.prototype.constructor==Person); //false
          alert(Person.prototype.constructor==Object); //true

如果想要让它重新指向Person，可以显示的进行赋值

         function Person(){  }
         Person.prototype={
             constructor:Person,
             name:"Tom",
             age:29
          }
          var p1=new Person();
          alert(Person.prototype.constructor==Person); //true
          alert(Person.prototype.constructor==Object); //false

4,prototype应用实例

计算数组最大值:

        var str=[10,50,30,40,20];
        function getMax(){
           return Math.max.apply(window,this); //apply内参数，前一个是对象，第二个是数组
       }
        Array.prototype.getMax=getMax;  //添加原型链，所有数组都可以共享getMax
        var max=str.getMax();
         console.log(max);
call，apply借用，一般在对象里用的比较多
        var a=[10,20,30,40,50];
        var max= Math.max.apply(this,a);  //此处的this可以换成window,结果一样
        alert(max);
